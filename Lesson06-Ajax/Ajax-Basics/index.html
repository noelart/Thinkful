Ajax

To build the data-driven apps that users increasingly expect and take for granted, you need the ability to make numerous server calls after a template webpage has loaded. AJAX is mostly used for these instances. AJAX stands for asynchronous JavaScript and XML

AJAX allows you to build web pages which make multiple calls to multiple servers without loading a new page. Specifically AJAX requests data from servers and then make that data accessible in the DOM.
