API

API stands for application programming interfaces. APIs allow you to use AJAX to request and display content from other databases, such as those of YouTube and Instagram. Many companies nowadays provide access to online databases by distributing an API. A website can show grpahs driven by this data.

        
          
Accessing an API with JSON

          omdbapi.com/?s=Blade%20Runner&r=json
