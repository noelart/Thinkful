Javascript basics - Strings

Introduction to strings

A string is a collection of characters that is used to represent text. You can create a string by enclosing in single or double quotes. Starting and ending delimiters must match.

          
            var story = '"This is what I meant by success" said the developer on completing the app right on his work\'s deadline';
            console.log(story);

Concatenation

Numbers, variables and strings can be concatenated together with a + returning a new string. Similalry using += new strings canbe added to original string variable.

          
          var string = "This is a number concatenated to a string" + 22;
          console.log(typeof string); string

          var message = 'Hello';
          message += ', Pierre';
          console.log(message); 'Hello, Pierre'

parseInt() and parseFloat()

A data-type string with a number can be converted to an integer with parseInt() or a float with parseFloat()

        
          var intFromString = parseInt('45k');
          console.log(typeof intFromString); number

Substrings

substring(arg1, arg2) takes two arguments: arg1 substring begins and arg2 where it ends. Second arg is optional - it defaults to the end of the string when not declared.

        
          var message = 'Hello World!';
          var firstString = message.substring(0, 5);
          var secondString = message.substring(6, 11);
          var worldMark = message.substring(6);

          console.log(firstString + '\n' + secondString + '\n' + worldMark); Hello World World!

indexOf() lastIndexOf() charAt()

Stign methods to locate the index of a substring or a character

  • indexOf('substring') locate the first occurance of a substring
  • lastIndexOf('substring') locate the last occurance of a substring
  • charAt(index) locates the character at a particluar index
          
          var searchTerm = 'San Francisco';
          var address = 'Hotel San Francisco, 123 Market Street, San Francisco, CA';
          var cityIndex = address.indexOf(searchTerm);
          var cityChar = address.charAt(44);
          var cityLastIndex = address.lastIndexOf(searchTerm);

          console.log('=====================' + '\n' + cityIndex + '\n' + cityChar + '\n' + cityLastIndex); 6 F 40
          
        
          
            var stringBetweenMatches = function(searchString, string) {
              var startString = string.indexOf(searchString) + searchString.length;
              var lastString = string.lastIndexOf(searchString);
              var stringMatch = string.substring(startString, lastString);
              return stringMatch;
            };

Comparison operator

Checking for mathcing strings witha comparison operator.

  • == will check if two strings are the same. If strings are the same returns true.
  • != will check if two strings are not the same. If strings are not the same returns true.
