Variables

Definition

Values can be assigned to variables so they can be referred to in code more easily. The ability to compute values, assign these values to variables and store away these variables for later use is a huge part of programming.

  • The value for a variable has always to be one of the Javascript data-types
  • A variable is a reference to a location in memory
  • Once a value is stored in a variable, we can sue this variable in other functions or expressions. 
                  
                var now = new Date();
                var month = now.getMonth();
  • The value of the variable can be changed at any point, the last assigned value will be the only one valid. 
                    
                  var temperature = 17;
                  console.log(temperature); 17

                  temperature = 15;
                  console.log(temperature); 15

                  temperature -= 5;
                  console.log(temperature); 10
