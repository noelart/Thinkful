Conditional programming

Introduction

A coditional is a statement which resolves to either true or false:

  • If Condition is true: var === value. When conditon is met the flow of control returns or assigns a value or calls a function. 
                  
              if (value === 5) {
                  return correct;
              } else {
                  return wrong;
              }
  • If Condition is false: var != value 
                  
              if (city != 'Sydney') {
                  return 'London';
              } else {
                  return 'Sydney';
              }

When testing a condition for a numeric value always use strict equality === as in

            
              if(value === 5){}

When testing for a condition when the value is not eqaul to use !==

Chained conditionals

Conditions can be chained together starting with if, followed by one or more else if and finish with else, normally used as the default conditional.

          
            var canIGet = function(money, item) {
              var items = ["MacBook Air", "MacBook Pro", "Mac Pro", "Mac mini"];
              var purchase = null;

              if ((item == items[0]) && (money >= 1200)) {
                purchase = true;
                } else if ((item == items[1]) && (money >= 1299)) {
                  purchase = true;
                } else if ((item == items[2]) && (money >= 2499)) {
                  purchase = true;
                } else if ((item == items[3]) && (money >= 500)) {
                  purchase = true;
                } else {
                  purchase = false;
                }
              return purchase;
            };

            console.log(canIGet(2000, 'MacBook Pro'));

Testing for truthiness

The notions of truthy and falsy are used to capture this idea that there are values other than the booleans true and false that nevertheless evaluate to true or false in conditional statements.

A list of falsy values:

  • Boolean value FALSE
  • empty string ""
  • number 0 sometimes this can be a disbaled state of an html element
  • variable not assigned null
  • variable not defined undefined
  • variable not a number data-type NaN

A list of truthy values include:

  • Boolean value TRUE
  • number 1 sometimes this can be an enabled state of an html element.
  • Empty arrays []
  • Objects {}
  • Functions function(){}
  • Strings matched if (searchString == "London')
  • Expected NOT variable if(!Object.create) mostly used to initialize an object, eg. a username

FizzBuzz challenge

          
            function fizzBuzz() {
            var fizz = 'Fizz';
            var buzz = 'Buzz';
            var array = [];
            var string = null;

          // Loop
            for (var i = 0; i <= 100; i++) {
              if ((i % 3 === 0) && (i % 5 === 0)) {
                string = fizz + buzz;
                array.push(string);
              }
              else if (i % 3 === 0) {
                string = fizz;
                array.push(string);
              }
              else if (i % 5 === 0) {
                string = buzz;
                array.push(string);
              }
              else {
                array.push(i);
              }
            }
            return array;
          }

          console.log(fizzBuzz());

Geography quiz

          
            var countries = ['England', 'France', 'Germany', 'Italy', 'Spain', 'Denmark'];
            var whichCity = function(country) {
              var city = null;
              if (country == 'England') {
                city = 'London';
              } else if (country == 'France') {
                city = 'Paris';
              } else if (country == 'Germany') {
                city = 'Berlin';
              } else if (country == 'Italy') {
                city = 'Rome';
              } else if (country == 'Spain'){
                city = 'Madrid';
              } else {
                city = "City not listed";
              }
              return city;
            };

            var listOfCities = function(array) {
              var output =[];
              for(var i = 0; i < array.length; i++) {
                var city = whichCity(array[i]);
                output.push(array[i] + ': ' + city);
              }
              return output;
            };

            console.log(listOfCities(countries));
