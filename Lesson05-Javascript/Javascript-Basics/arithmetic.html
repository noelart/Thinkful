Numbers, Arithmetic Operators and Comparators

Numbers

Numbers are one of the primitive types defined in JavaScript.

  • Five arithmetic operators in JavaScript:
    • mulitplication
    • division
    • addition
    • subtraction
    • modulus
  • Operator precedence refers to teh order of execution is based on PEMDAS:
    • Parenthesis
    • Exponents
    • Multiplication/Division
    • Addition/Subtraction
  • Modulus operator performs an implied division and returns the remainder.
    •             
            var modulus = function(num1, num2) {
              return num1 % num2;
            };
             Output:
            console.log('Modulus: ' + modulus(50, 4)); Modulus: 2

Applying ternary operator

The ternary operator serves as an alternative to the if statement 

          
            <variable> = <expression> ? <true clause> : <false clause>
  1. Does <expression> evaluate to true, or does it evaluate to false?
  2. If <expression> evaluates to true, then the value of <true clause> is assigned to <variable>, <false clause> is ignored, and the next statement is executed.
  3. If <expression> evaluates to false, then <true clause> is ignored and the value of <false clause> is assigned to <variable>.

Protoype for arithmetic functions

A sum prototype with various arithmetic functions

        
        var Sum = function(num1, num2) {
          this.num1 = num1;
          this.num2 = num2;
        };

        Sum.prototype = {
          add : function(num1, num2) {
            return num1 + num2;
          },

          subtract : function(num1, num2) {
              var output = (num1 > num2) ? (num1 - num2) : (num2 - num1);
              return output;
          },

          multiply : function(num1, num2) {
            return num1 * num2;
          },

          divide : function(num1, num2) {
            var output = (num1 > num2) ? (num1 / num2) : (num2 / num1);
            return output;
          }
        };

        var sum = new Sum();
        console.log(sum.add(11, 44));
        console.log(sum.subtract(73, 33));
        console.log(sum.multiply(10, 20));
        console.log(sum.divide(20, 400));

Comparators

The comparison operators are available for comparing any two data types:

  • > greater than
  • < less than
  • == equality
  • === strict equality
  • != not equals
  • !== strict not equality
  • >= greater or equal to
  • <= less or equal to
