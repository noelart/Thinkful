Data types

JavaScript is classified as a prototype-based scripting language with dynamic typing and first-class functions. It has Type inference meaning it can auto deduct the data type if a number is a string or a number. Or a number is a float or an int.

Numbers

Javascript only have one type of numbers: 64-bit floating unit. Integer values in JavaScript can run from negative to positive 9,007,199,254,740,992, and decimal values can get as small as 5x10-324.

  • Values larger than the integer limit are represented by ∞.
  • Values which are not numbers but are String-type are represented by NaN
          
            var sum = function() {
            return 12 / 'hello';
            };
            Output:
            console.log(sum()); NaN

            var num = 22;
            Output:
            console.log(typeof num); number

Strings

Strings are an array of alphanumeric characters

          
            var name = "Pierre";
            Output:
            console.log(typeof name); string

Booleans

Boolean value can either be TRUE or FALSE

          
            var showValue = true;
            Output:
            console.log(typeof showValue); boolean

Functions

Are javasciprt objects that perform an action or return a value.

  • Anonymous functions
  • Named functions
          
            var sayHello = function(name) {
              return 'Hello ' + name;
            };
            Output:
            console.log(sayHello('Scott')); Hello Scott

Objects

Objects are a composite Javascript object combinig properties and methods - functions which are an object porperty.

          
            Object literal:
            var person = {
              firstName : 'Noel',
              secondName : 'Sydeny',
              age : 22
            };
            Output:
            console.log('First name :' + person.firstName + '\n' + 'Second name: ' + person.secondName + '\n' + 'age : ' + person.age);
            First name :Noel
            Second name: Sydeny
            age : 22

Null

Null is used to assign a placeholder value to a variable when it has no other value. It signifies the absence of a specific value.

          
            var valueNotAssigned = null;
            Output:
            console.log(typeof valueNotAssigned); object

Undefined

A variable that is declared but that doesn’t have a value assigned to it. A space in memory is reserved for it.

          
            var valueNotDefined;
            Output:
            console.log(valuNotDefined); undefined
