Functions

Functions are objects which store behaviours. Two types of functions:

  • Named functions are those with a name assigned to it before the brackets. 
              
            function name(arg1, arg2){}
            function addSum(num1, num2) {
              return num1 + num2;
            };
  • Anonymous functions are functions with no name assigned before the brackets. 
              
          function(){}
            var total = function(num1, num2) {
              return num1 + num2;
            };

Here's an interesting blogpost with some functions basics noahstokes.com. Functions are useful for re-using code by employing a function call. This is a standard approach in programming, referred to DRY (Do not Repeat Yourself)

Scope

Scope refers to the context were variables or fucntions are accesible. A code loop from 1 to max has var i as a local var. This var is not accessible outside the loop.

              
                var i as a local variable
                function fizzbuzzLong(max) {
                var string = "";
                var array = [];
                for (var i = 1; i <= max; i++) {
                 if ((i % 5 === 0) && (i % 3 === 0)) {
                  string += 'FizzBuzz' + ', ';
                 } else if (i % 3 === 0) {
                  string += 'Fizz' + ', ';
                 } else if (i % 5 === 0) {
                  string += 'Buzz' + ', ';
                 } else {
                    string += i + ', ';
                 }
                }
                return string;
              }

Closures

Is a the parent function() with a nested child function(). Closures return a value. In the example below locally-scoped function check(num) returns a string (either 'Buzz' or 'Fizz'). check(num) cannot be called outside the fizzbuzzer() block.

        
        Closure returning a value
          function fizzbuzzer(max) {
              var array = [];

              function check(num) {
                var string = '';
                if (num % 3 === 0) { string +='Fizz'; }
                if (num % 5 === 0) { string +='Buzz'; }
                return string || num;
              }

              for (var i = 1; i <= max; i++) {
                array.push(check(i));
              }
              return array;
            }

            console.log(fizzbuzzer(30));

Closures can return a function() as well. In the example below the closure returns a function that takes an other argument. The returned function is avalibale outside the scope of the closure.

        
          function adder(x) {
            return function(y) {
              return x + y;
            }
          }

          var add3 = adder(3);
          var add5 = adder(5);

          console.log(add3(5)); 8
          console.log(add5(5)); 10
