jQuery Link Layover

Links can be used to call jQuery functions such as showing or hiding content.

  • jQuery showing-hiding events:
  • .fadeIn() makes contetn visible. Can take slow or fast as arguments.
  • .fadeOut() makes content disappear.
  • .fadeToggle() shifts between fadeIn() and fadeOut()
        
        css style to start by hiding the content
        .comments { display : none; }
