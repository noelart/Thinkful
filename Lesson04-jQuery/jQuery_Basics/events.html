jQuery Events

  • slideDown() shows an element. The element would have a css style of display: none;
  • slideUp() applies display: none; to an element.
  • slideToggle() switches between slideDown() and slideUp() everytime the event-handler is called.
          
              $('#tour').on('click', 'button', function() {
                $('.photos').slideToggle();
              });

on() events

Event-handlers can be hoisted in their own anonymous function and called when needed in the on() jQuery function. Fucntion is called immedaitely if it is passed as an argument as follows showTicket()

          
            Anonymous Function
            function showTicket() {
            $(this).closest('.confirmation').find('.ticket').slideDown();
            }

            Function called by jQuery when event triggered
            $('.confirmation').on('click', 'button', showTicket);
            Function called by jQuery immediately
            $('.confirmation').on('mouseleave', 'h2', showTicket());
