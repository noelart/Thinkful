Traversing and searching the DOM

Traversing

Using specific elements is faster to access the DOM than using .find(), .prev() and .next()

Method chaining is the ability to call multiple jQuery methods in a row on a single object. This is possible because every call to a jQuery method returns the jQuery object that it operated on.

  • Parent <ul> element: transport routes
    • mountain route
    • sea route
    • land route
    • air link
        
         $('.routes li').first().css({'color' : '#CC2244'});
         $('.routes li').last().prev().css({'color' : '#008800'});

Method chaining to access parent or chidlren using .children(), .parent()

        
          $('.routes').parent().prev().css({'list-style' : 'none', 'text-transform' : 'uppercase'});

Method chaining using .find() will select all children elements within that class
